如何准备实习求职并拿到微信offer

发表于 2020-02-04 更新于 2020-02-08 分类于 求职 阅读次数： Valine：

人们常说读十几年的书就为了找个好工作，这种求职压力是不言而喻的。2018年的春天是我研究生阶段最黑暗的一段时光，努力三个月换来一次次的面试失败，以至于担心将来会没有工作，犹豫要不要换方向、要不要回归机械老本行。但胜利是属于坚持者的，五月开始我接连拿到头条、百度的实习offer，六月拿到微信的实习补招offer。
时至今日，在微信工作了快一年，希望把后台开发的学习经验分享给大家。

提早打好基础

秋招的时候最好有实习经历，找实习之前最好已经打好扎实的基础。

目前冯·诺伊曼体系的计算机仍然是主流，由此而衍生的计算机系统、网络、存储（数据结构）是必须要打好的基础。掌握好这些基础是很重要的，将来实际项目的性能提升全靠它们，这也是区分好的程序员的标准。

图 计算机与网络体系

计算机系统

书单：

  • 深入理解计算机系统
  • UNIX环境高级编程
  • 鸟哥的Linux私房菜

一般除了工业领域、银行、事业单位会用windows server，其他都用的linux后台，所以linux的常用命令、基本操作是必会的。另外计算机系统的基础还会考察并发编程，进程、线程的地址空间，死锁，进程间通信，缓存淘汰，异步调用。如果有实际项目，最好能讲出曾解决过coredump，内存泄漏，内存碎片等问题。

网络

书单：

  • TCP/IP详解 卷1：协议
  • UNIX网络编程 卷1 套接字联网API

网络主要考察的是通讯协议和套接字。TCP/IP中的网络层和应用层比较重要，网络层要掌握三次握手、四次挥手的细节，应用层主要是http和https，https里面的tls原理。进阶一点需要掌握网络拥塞、粘包。

socket套接字考察的是实际编码。需要掌握进程池、线程池网络模型，poll、epoll，异步、同步、阻塞、非阻塞I/O模型。最好懂一点常见的网络攻击与防御。

存储

书单：

  • Redis设计与实现
  • 高性能MySQL

存储是后台开发中很重要的一部分，推荐kv存储和关系型存储的两个代表：redis和mysql。其中redis只能算是缓存，但对于理解kv存储架构有很大帮助。进阶一点需要了解下高并发和容灾，可以看下leveldb的实现、paxos协议、raff协议等。

编程语言

书单：

  • C++ Primer
  • Effective C++
  • STL源码剖析

Java我使用的不多，这里主要推荐下C++的该准备哪些。C++ Primer这本书每一章都很重要，特别是面向对象和stl部分。面向对象需要知道一个类的内存空间，虚函数的实现原理。stl需要知道常见容器的底层实现。另外单例的实现和c++11（c++17）的新特性等也需要了解。

刷题进入状态

视频：
网易公开课#麻省理工学院公开课：算法导论

书单：

  • 算法(第4版) （java）
  • 算法导论（原书第3版） （c++）
  • 剑指Offer

程序就是数据结构和算法，算法与数据结构是不可分割的，比如排序会用到红黑树，哈希会用到跳跃表。如果基础打得好，数据结构应该没有问题，下一步就要提高算法水平和解题速度。除了按照上面视频和书单提高算法的理论知识外，还需要适量刷题，leetcode刷到中等难度就够了。刷题不在于量，找几个典型的研究透解题套路即可。

多面多总结

准备了这么多，接下来就要迈出第一步去公司面试。准备面试时可以上牛客网看看这个公司的面试形式、难度，考察重点是什么，会问哪些问题。技术岗位的面试一定要实事求是。需要注意的是，面试是一个双方互相了解的过程，你的面试官很有可能就是你将来的leader，你的部门就是面试官所在的部门，可以抓住机会了解你将来的工作环境。

笔试和面试的总结是很重要的，很有可能这次面试没答出来，错过这家公司，下次面试又被问到同样的问题。

外企、私企、国企？

在我看来，

  • 外企比较守法规，一般不加班，加班按国家标准发补贴，薪资相对私企低一点，技术能力会有提高；
  • 私企去的人最多，一般加班比较多，995甚至996是常态，薪资水平较高，技术能力会有提高；
  • 国企、银行一般只招应届生，薪资低，技术成长不高，工作稳定轻松。
    薪酬是一个很重要的指标，可以关注浙大人出的校招薪水公众号了解各个公司各岗位的薪资待遇。但薪资只是其中一方面，将来工作的幸福感还取决于工作环境和自我实现。个人认为，私企、外企能培养能力，只要自身技术过硬，不论环境怎么变化都不会丢饭碗。

读研or工作

本科生在大三时会碰到抉择，选择读研还是直接工作，个人结论是建议读研。

读研：

  • 思维深度和广度会有提高，能系统性看待问题；
  • 延续校园时光，认识更多的老师、同学，甚至找到女朋友；
  • 找工作上面学历高更有优势。但本科生优势在于年龄小，互联网行业年龄越小越有潜力。

微信工作感悟

在腾讯微信事业群工作快满一年，在工作环境、福利待遇、个人提升方面谈下我的看法。

  • 工作环境比不上微软、谷歌，但在国内还算可以，工作时间大概10-10-5，同事们、领导们也都比较务实好相处，80后和90后会有点代沟；
  • 福利待遇在业内也还算可以，但不算高。千万不要被HR发offer时给的报价给迷惑了，其实把股票、房补、签字费、各种补贴算进去了，实际到手的工资并没有那么高；
  • 大公司培训机会比较多，内网也有非常多的干货可供学习；大公司不缺人才，大牛到处都是，个人晋升压力比较大。而且干部岗位有限，可能只是要干个十年、八年才能升到组长；
  • 大公司里面核心部门和非核心部门的差距确实存在，边缘部门的资源投入、技术能力和福利待遇就是要弱一些。

实验室师兄把学习路线传授给我，我又把经验带给学弟，同样非科班的学弟已经拿到广州欢聚时代的offer，现在我把我的所得分享给大众，希望对你们有点帮助，欢迎在下方评论区留下你的想法~
